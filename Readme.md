## Descripción
Este proyecto corresponde al **Desafío 7 – API FutScript**, desarrollado como parte del Módulo 6 del curso Fullstack JavaScript.  
El objetivo es implementar una **API RESTful con Node.js** para la gestión de equipos y jugadores de fútbol, incorporando **autenticación mediante JWT**, conexión a una base de datos PostgreSQL y pruebas automatizadas.

---

## Tecnologías utilizadas
- Node.js  
- Express.js  
- PostgreSQL  
- pg  
- JSON Web Token (JWT)  
- Jest  
- Supertest  
- JavaScript  

---

## Logros

- [x] Implementación de **autenticación y autorización de usuarios mediante JWT**.
- [x] Uso del paquete **pg** para la comunicación y gestión de una base de datos PostgreSQL en una aplicación Node.js.
- [x] Ejecución de **consultas SQL** para la obtención e inserción de datos desde el servidor.
- [x] Uso correcto de los **códigos de estado HTTP** según el resultado de cada operación.
- [x] Implementación de **tests unitarios** utilizando los paquetes **Jest** y **Supertest** para validar el funcionamiento de la API.
